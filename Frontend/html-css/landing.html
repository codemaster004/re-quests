Technologies we used

Front-end

On frontend we are using Java Script framework Vue for seamless transitions betwean pages and close to instant data loading. As usually website itself is build using html and css and with mobile first approach so it shoud be perfectly scaleble.

Back-end

For backend we creaeted REST API using C#. We store user data inside PostgreSQL Databse. It cominicates with our frontend using JSON and carefully picked request codes. A few more than 404, 500 and 200. :)

Design

We used Figma app for UI and UX design. We also generated 3D objects to create awesome medal gifs. All of the icons and logo were drawn on the Ipad pro (show of) in Procreate app.
